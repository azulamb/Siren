セイレーン作戦進捗確認ツール - データのインポート

このツールはCSVによるデータのエクスポートとインポートに対応しています。

手順

手順は以下のようになります。

  1. ユーザーデータをダウンロードする
  2. ダウンロードしたCSVを編集する
  3. ユーザーデータをインポートする

1. ユーザーデータをエクスポートする

ユーザーデータをダウンロードしてください。CSV形式で現在のデータを出力することが出来ます。

2. ダウンロードしたCSVを編集する

編集にはいくつか方法があります。半角英数字しか使っていないためWindowsであればメモ帳などのテキストエディタでも可能ですが、表計算ソフトを使うことをおすすめします。

Google Spreadsheetを使う

  1. Google Spreadsheetを開きます。
  2. 新しい空白のSpreadsheetを作ります。
  3. 左上のメニューの「ファイル」→「インポート」でインポート画面を開き、「アップロード」のタグを選択します。
  4. ダウンロードしたCSVをアップロードします。
  5. デフォルトの設定のままデータをインポートします。
  6. 編集した後左上メニューの「ファイル」→「ダウンロード」→「カンマ区切りの値（.csv、現在のシート）」をクリックして編集後のユーザーデータをダウンロードします。

これでデータのインポートが出来ました。「mission_1」などと書かれた列が入力する値で、「max_1」などと書かれた列が任務の最大値となっています。

（ユーザーデータ単体で進捗管理できるよう現在値と最大値を一緒に出力しています。）

後はセイレーン作戦の海域探索のところから得られるデータを入力していきます。

具体的には「11 カリブ海」（番号が11）の「異常気象区域を調査する」（任務の上から2番目）が「10」だったときはCSVでのnoが「11」の行の「mission_2」の列の値を「10」に書き換えます。
